# Billionaire Simulator

A fully functional, client-side billionaire simulation game built with React, Vite, and Tailwind CSS.

## Features

- **Multi-Wallet System**: Manage 5 distinct wallets (SimCash, PaySim, UltraWallet, CryptoSim, BankSim).
- **Global Economy**: Shop across 6 different marketplaces with unique currencies and mechanics.
- **Passive Income**: Build businesses and assets that generate income even while you idle.
- **Rank Progression**: Climb the ladder from Hustler to Trillionaire.
- **Persistent Save**: Your progress is automatically saved to your browser's local storage.

## Tech Stack

- **Framework**: React + Vite (TypeScript)
- **Styling**: Tailwind CSS
- **State Management**: React Context API + useReducer
- **Icons**: Lucide React
- **Animations**: CSS Transitions + Tailwind

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Deployment on Vercel

This project is optimized for deployment on Vercel.

1.  Push this code to a GitHub repository.
2.  Log in to [Vercel](https://vercel.com).
3.  Click "Add New Project" and select your repository.
4.  Vercel will automatically detect the Vite framework settings:
    - **Build Command**: `npm run build`
    - **Output Directory**: `dist`
    - **Install Command**: `npm install`
5.  Click "Deploy".

That's it! Your Billionaire Simulator is now live.

## Disclaimer

This is a simulation game. No real money is involved. All currencies and transactions are fictional.
